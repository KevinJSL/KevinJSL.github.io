Billiard Ball Simulation in Unreal Engine 5

Physics-based simulation project by Kevin Land

Project Overview

This project simulates billiard ball dynamics in Unreal Engine 5 using a custom C++ Actor class that allows all balls to share and access each other's state variables. Each ball is a textured sphere, and collision and frictional behavior are fully simulated using physical parameters. Blueprints manage the overall flow of the simulation, while C++ handles precise motion and collision responses.

Pool Table Setup

Friction and Restitution Tests

Each test run adjusts surface friction coefficients (us, ur) and restitution e to visualize how the balls behave under different physical parameters.

  • Basic: ur = 0.01, us = 0.2, e = 0.7
  • High Friction: ur = 0.12, us = 0.8, e = 0.7
  • Low Friction: ur = 0.0, us = 0.0, e = 0.7
  • Low Restitution: ur = 0.1, us = 0.2, e = 0.1
  • High Restitution: ur = 0.1, us = 0.2, e = 1.2
  • Experimental: Gain velocity per hit (eventually breaks collision)
