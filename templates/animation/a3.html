Dynamic Particle System in Unreal Engine 5

Assignment write-up by Kevin Land

Overview

This project expands upon a basic cone-particle generation system by combining Blueprint logic with a custom C++ Actor Component. The goal was to recreate and extend the standard particle emission behavior from the assignment write-up, while adding parameters for visual and dynamic control such as color, gravity, and size interpolation. I used the keyframing from assignment 1 for movement.

1. Naïve Blueprint Solution

I began with a Blueprint-only approach to generate and animate cone particles. This version was functionally correct but fairly rigid — it lacked flexibility and required a large, repetitive node graph to tweak basic properties.

2. C++ Actor Component

I then refactored the core logic into a C++ Actor Component that handles particle generation, emission direction, and parameter control. This C++ component reproduces the standard cone-emission behavior while exposing several modifiable properties to Blueprints.

  • Base cone emission pattern (radius, spread, and velocity)
  • Adjustable gravity and size interpolation
  • Color interpolation (from green → red)
  • Randomized color mode for dynamic effects

3. Customization Features

The following parameters are adjustable directly from the component, allowing flexible behavior during runtime or via Blueprint scripting:

Gravity & Size Interpolation

Gradually modifies particle gravity and scale over lifetime. This gives the motion a more realistic lift-and-fall dynamic.

Color Interpolation

Smoothly transitions colors from green to red based on lifetime or speed.

Randomized Colors

A favorite experiment — randomizing RGB components at spawn time creates a lively, energetic effect that’s great for explosion or magic-type visuals.

5. Reflection

This experiment showed how combining C++ components with Blueprint scripting enables much greater flexibility. While the Blueprint prototype was limited and visually rigid, the C++ version allows parameterized behaviors and reusable effects. The random color generation in particular was an unexpected but fun visual improvement.
